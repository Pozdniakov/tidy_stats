## 🔴 Критичные

1. **[строка 41]** `Результат очень похож по своей структуре на `glm()`` → Опечатка: должно быть `lm()`. Сравнивается вывод GLM с выводом LM, а не GLM с самим собой. **(2/3)**

2. **[строка 54]** `Модели со смешанными эффектами позволяет включать` → Грамматическая ошибка: «Модели... позволяют» (множественное число). **(2/3)**

3. **[строка 90]** `одна из которых является "вложенной" в другую, то есть усложненной версией более общей модели. В данном случае, более общая модель предполагает, что время реакции увеличивается у всех испытуемых одинаково` → Логическая инверсия: вложенная модель — это *упрощённая* (частный случай), а не «усложнённая версия более общей». Далее «более общая» применяется к модели `sleep_lme0`, которая на самом деле менее общая. Исправить: «одна из которых является вложенной, то есть упрощённой версией более сложной модели. Более простая модель предполагает...». **(2/3)**

4. **[строка 7]** `$$Y = XB$$` → Формула общей линейной модели без члена ошибки. Стандартная запись: $Y = XB + E$. То же на строке 17. **(1/3, верифицировано)**

5. **[строка 11]** `![](https://lindeloev.github.io/tests-as-linear/linear_tests_cheat_sheet.png)` → Изображение без alt-текста (нарушение §23). Также внешний URL — при недоступности ресурса изображение пропадёт. Рекомендуется скачать в `images/` и добавить alt-текст. **(2/3)**

6. **[строка 62]** `` ```{r, eval = FALSE} `` → Устаревший синтаксис чанка (нарушение §17). Заменить на `` ```{r} `` с `#| eval: false`. **(3/3)**

7. **[строка 67]** `library(lme4)` → Чанк без `#| message: false`. При загрузке `{lme4}` выводятся сообщения, которые нужно подавить. **(2/3)**

## 🟡 Важные

1. **[строка 34]** `еще один важный параметр `family =`` → По §8 это аргумент функции, не параметр. Заменить «параметр» на «аргумент». **(3/3)**

2. **[строка 60]** `` `nlme` и более современный `lme4` `` → Пакеты без фигурных скобок (нарушение §9). Должно быть `{nlme}` и `{lme4}`. **(3/3)**

3. **[строки 5, 15, 19, 41, 54]** Термины при первом введении не оформлены по §7 (**жирный** + *курсивный перевод*). Затронутые термины: общая линейная модель, обобщённая линейная модель (частично оформлены — перевод не курсивом), AIC / информационный критерий Акаике, фиксированные эффекты, случайные эффекты, иерархическая регрессия, многоуровневое моделирование, модели со смешанными эффектами. **(3/3)**

4. **[строка 54]** `много названий - в зависимости` → Дефис вместо длинного тире. Должно быть `---`. **(2/3)**

5. **[строка 41]** `Adjusted R^2^` → Markdown-суперскрипт вместо LaTeX. В той же строке уже используется `$R^2$`. Заменить на `$R^2_\text{adj}$` или `скорректированный $R^2$` для единообразия. **(3/3)**

6. **[строка 41]** `это критерий использующийся для выбора` → Пропущена запятая перед причастным оборотом: «критерий, использующийся для выбора». **(1/3, верифицировано)**

## Резюме

Глава компактная (127 строк) и охватывает три важные темы — общую линейную модель, обобщённую линейную модель и смешанные эффекты. Главные проблемы: (1) опечатка `glm()` вместо `lm()` на строке 41, искажающая смысл; (2) логическая инверсия «вложенная = усложнённая» на строке 90, запутывающая читателя; (3) систематическое отсутствие форматирования терминов по §7 при первом введении. Глава также нуждается в итоговом абзаце и связующем переходе к разделу о смешанных моделях. Формулы Y=XB стоит дополнить членом ошибки.
